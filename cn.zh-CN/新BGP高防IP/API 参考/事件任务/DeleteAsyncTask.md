# DeleteAsyncTask {#reference363 .reference}

调用DeleteAsyncTask删除指定的异步任务。

## 请求参数 { .section}

|名称|类型|是否必需|描述|
|--|--|----|--|
|TaskId|Long|是|要删除的任务ID。|

## 返回参数 { .section}

|名称|类型|描述|
|--|--|--|
|RequestId|String|本次请求的ID。|

## 示例 { .section}

**请求示例**

```
{
  "TaskId": 1
}

```

**返回示例**

```
{
  "RequestId": "0bcf28g5-d57c-11e7-9bs0-d89d6717dxbc"
}

```
